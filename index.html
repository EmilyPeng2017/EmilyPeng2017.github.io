How to use hexo to create page, tag and category

Posted on 2022-10-09 In Tools

Once setting up the hexo environment, it’s time to create your new pages! Shall we just use mkdir cmd to create subfolders under source folder? The answer is NO. Please realize this is HEXO! We could use “hexo cmd” directly!

1. create new file

  • cmd
    hexo new <title>

the file would be generated under hexo/source/_posts folder

  • settings

2. create new tag

  • cmd
    hexo new page tags

  • settings

3. create new category

  • cmd
    hexo new page categories

  • settings

for more information, please go to hexo website
